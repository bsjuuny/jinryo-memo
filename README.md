# 진료메모

병원 방문 전 증상과 질문을 짧게 정리해 의사에게 전달할 내용을 준비하는 프론트엔드 MVP입니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 엽니다.

## 주요 기능

- 증상 입력: 대상, 나이, 시작 시점, 주요 증상, 체온, 복용한 약, 동반 증상, 생활 상태, 질문, 메모
- 규칙 기반 진료용 요약 카드 생성
- 요약 복사
- 인쇄하기
- localStorage 기록 저장, 다시 보기, 삭제
- 고열, 호흡곤란, 의식저하, 경련, 탈수 의심 등 표현 포함 시 의료진에게 빠르게 문의하라는 안전 안내 표시

## 안전 원칙

- 진단, 처방, 치료 판단을 제공하지 않습니다.
- 질병명을 추정하지 않습니다.
- 약 복용을 지시하지 않습니다.
- 응급 여부를 단정하지 않습니다.
- 이 앱은 병원에서 설명할 내용을 정리하는 진료 전 메모 도구입니다.

## 구조

- `src/types/visitMemo.ts`: `VisitMemo` 타입
- `src/lib/summary.ts`: 규칙 기반 요약 및 안전 표현 감지
- `src/lib/storage.ts`: localStorage 저장소
- `src/components/*`: 입력, 요약, 기록, 안전 안내 컴포넌트
